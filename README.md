Проект состоит из двух основных компонентов:

### Серверная часть
- Реализован на **C++20** с использованием Crow  
- REST API для обработки HTTP-запросов  
- Обработка регистрации, авторизации и команд от клиента  
- Возможность интеграции с базой данных и безопасного хранения данных через libsodium  
- Поддержка JSON (встроенные возможности Crow)

### Клиентская часть (Qt)
- Графический интерфейс для регистрации, авторизации и управления устройствами  
- Подключение к серверу через HTTP-запросы  
- Отправка команд и получение ответов  
- Кроссплатформенность: Linux, macOS, Windows  

---

## Технологии

- 🖥 **C++20** — современный язык для серверной логики  
- 🌐 **Crow** — лёгкий HTTP-фреймворк для создания REST API  
- 💻 **Qt** — GUI и клиентская логика  
- 🔒 **libsodium** — криптография и безопасное хранение данных  
- 🗄 **PostgreSQL** — планируется для хранения пользователей и сессий  

---

## Возможности

- Регистрация и авторизация пользователей через Qt-приложение  
- Взаимодействие клиент–сервер через REST API  
- Управление удалёнными устройствами  
- Масштабируемый сервер для подключения нескольких клиентов
